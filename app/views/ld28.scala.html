You only have one vote

You only have one vote is a LD28 Compo entry, with the theme being "You only have one". In You only have one vote, you are given a list of candidates to be the next world leader. Each candidate has a set of policies they want to implement when they are elected. Under each policy is a graph showing how this will affect society (Gray=Neutral, Red=Negative effect, Green=Positive effect). Using this supplied data you must pick the candidate most likely to succeed. You can only vote once, so make a good decision.

Once you vote, a simulation will pop up showing how each policy affects hundreds of virtual citizens. This is shown by the "Society bar", if the marker is in the red zone, your leader isn't doing too well. If your leader is in the green his policies are doing well. Once the game ends you can refresh and hundreds of citizens will be regenerated + new candidates will be generated. Have fun! (And don't forget to vote for my entry on LudumDare!)
